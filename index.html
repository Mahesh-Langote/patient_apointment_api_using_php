Patient Appointment System API Documentation

Doctor Listing Endpoint

URL: http://yourdomain.com/api/doctors

HTTP Method: GET

Description: Retrieves a list of all doctors.

Usage:

curl -X GET http://yourdomain.com/api/doctors

Doctor Detail Endpoint

URL: http://yourdomain.com/api/doctors/{doctor_id}

HTTP Method: GET

Description: Retrieves details about a specific doctor based on their `doctor_id`.

Usage:

curl -X GET http://yourdomain.com/api/doctors/{doctor_id}

Appointment Booking Endpoint

URL: http://yourdomain.com/api/appointments

HTTP Method: POST

Description: Allows users to book a new appointment. Users need to send a JSON payload with the doctor's ID, patient name, and appointment date.

Usage:

curl -X POST http://yourdomain.com/api/appointments -H "Content-Type: application/json" -d '{
   "doctor_id": 1,
   "patient_name": "John Doe",
   "appointment_date": "2023-10-10"
}'
